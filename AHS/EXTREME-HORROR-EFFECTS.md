## üîß CUSTOMIZE INTENSITY

### If TOO intense (computer slow or too scary):

**Reduce shake:**
```css
animation: extremeShake 0.15s infinite;  /* Slower */
/* Or remove from .scene.active animation list */
```

**Reduce glitches:**
```css
animation: extremeGlitch 4s infinite;  /* Less frequent */
```

**Less dark:**
```css
filter: brightness(0.7);  /* Brighter */
```

**Remove jump scares:**
```css
.scene.active::after {
    display: none;  /* No flashes */
}
```

### If STILL NOT ENOUGH (want MORE horror):

**Even darker:**
```css
filter: brightness(0.3);  /* VERY dark */
```

**More shake:**
```css
transform: translate(-10px, 10px) rotate(-2deg);  /* Bigger movements */
```

**Constant inversions:**
```css
.scene img {
    animation: extremeInvert 1s infinite;  /* Always inverting */
}
```

## üéì WHAT YOU'RE LEARNING

### Advanced CSS Techniques:
1. **Multiple animations** - Running 4+ animations simultaneously
2. **CSS filters** - contrast, brightness, saturate, grayscale, hue-rotate, sepia, invert
3. **Transform combinations** - translate + rotate + scale + skew together
4. **Pseudo-elements** - ::before, ::after for overlays
5. **nth-child selectors** - Targeting specific scenes (3n, 4n, 5n, even)
6. **Keyframe animations** - Complex multi-step animations
7. **Timing functions** - steps(), ease-in-out, linear
8. **SVG in CSS** - Embedded noise textures
9. **Gradients** - Radial and linear for vignette/scan lines
10. **Z-index layering** - Stacking multiple effects

### Horror Design Principles:
1. **Darkness = Fear** - Less visibility increases tension
2. **Unpredictability** - Random effects are unsettling
3. **Imperfection** - Glitches/grain suggest danger
4. **High contrast** - Creates dramatic mood
5. **Desaturation** - Removes life/warmth from image
6. **Movement** - Shake suggests instability
7. **Red** - Primal danger/blood association
8. **Distortion** - Reality breaking down
9. **Negative space** - Heavy vignette creates claustrophobia
10. **Rapid changes** - Jump scares work through surprise
